# set


## Usage

```
npx ember-v2-codemods set path/of/files/ or/some**/*glob.js

# or

yarn global add ember-v2-codemods
ember-v2-codemods set path/of/files/ or/some**/*glob.js
```

## Local Usage
```
node ./bin/cli.js set path/of/files/ or/some**/*glob.js
```
